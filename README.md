**Amazon Automation Testing Framework**
 **Overview**

    This project showcases an end-to-end automation testing framework built for the Amazon platform using Selenium WebDriver, TestNG, and Cucumber (BDD).
    The framework is designed to automate key user flows, ensure application stability, and support scalable test execution.

**Objectives**
   
    Automate critical user journeys of Amazon
    Implement BDD approach using Cucumber
    Enhance test reliability with TestNG
    Build a maintainable and reusable automation framework
    Generate detailed execution reports
**Features Automated**
  
    - Product Search
    - Add to Cart
    - Product Filtering & Sorting
 **Tech Stack**

        Language: Java
        Automation Tool: Selenium WebDriver
        Testing Framework: TestNG

**Key Highlights**
   
    BDD-based automation framework
    Reusable Page Object Model design
    Easy maintenance and scalability
    Clear reporting and logging system
**Conclusion**
    
    The automation framework effectively validates key functionalities of the Amazon platform, ensuring reliability and reducing manual effort through structured automation practices.
